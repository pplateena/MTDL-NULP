Лабораторна робота №3
Transfer learning у комп’ютерному зорі
Мета роботи. Опанувати стратегії роботи з моделями глибинного
навчання для задач класифікації зображень, візуалізувати процес навчання в
TensorBoard.
Завдання
1. Oзнaйoмитися з oснoвними тeoрeтичними вiдoмoстями тa
рeкoмeндoвaнoю лiтeрaтурoю зa тeмoю рoбoти.
2. Обрати CV‑датасет із >=2 класами, наприклад, медичні зображення,
Food101 або будь який інший довільний датасет.
3. Виконати розвідувальний аналіз даних (EDA), зокрема перевірити
кількість зображень, розподіл класів, розміри, за потреби варто застосувати
аугментації.
4. Обрати одну зі state‑of‑the‑art моделей із попереднім навчанням:
ResNet50, Vision Transformer або іншу.
5. Звести вхідні дані до того формату, який очікує обрана базова модель.
6. Заморозити всі або більшість шарів бази, аби зберегти загальні ознаки та
зменшити витрати часу/ресурсів. Додати новий класифікаційний head.
7. Підібрати оптимізатор, функцію втрат, batch size.
8. Тренувати лише новий head, залишивши базову модель нерухомою.
9. Запустити TensorBoard для моніторингу втрат і точності. Зберігати ваги
та графіки навчання.
10. Розморозити кілька верхніх шарів бази, які містять спеціалізовані
ознаки, і продовжити тренування з меншою швидкістю навчання.
Використовувати callback’и для ранньої зупинки та зменшення learning rate.
Знову логувати метрики в TensorBoard, порівнювати з попередньою фазою.
11. Повторити кроки 5–10 для іншої довільно обраної моделі.
12. Порівняти результати обох моделей та проаналізувати, чи достатній
розмір датасету для них обох.
13. Використати TensorBoard, щоб порівняти криві втрат/точності для
різних стратегій.
14. Побудувати градієнтні карти для кількох прикладів, щоб побачити, на
які області зображень модель звертає увагу. Описати якість інтерпретації.
15. Сформулювати висновки та зазначити, що вплинуло на точність
(кількість розморожених шарів, вид оптимізатора, модель), які проблеми
виникали, порівняти моделі та проаналізувати, чому виникли відмінності.
16. Оформити звіт. Структура звіту: титульний аркуш, мета роботи,
покроковий опис ходу роботи, приклад програмної реалізації, скріншоти з
демонстрацією результатів виконання програми, висновки.
Теоретичні ресурси
1. CS231n: Convolutional Neural Networks for Visual Recognition (Stanford) —
класичне пояснення transfer learning і fine-tuning.
2. Hugging Face Course: Chapter on Transfer Learning — на прикладах PyTorch
та Transformers.
3. Fine-tuning torchvision models (PyTorch) — офіційна інструкція.
4. Transfer Learning Guide (TensorFlow) — туторіал із використанням Keras
API.
5. Visualizing Models, Data and Training with TensorBoard — документація з
прикладами.
6. How to Fine-tune Vision Transformers — документація ViT у Hugging Face.
7. ONNX: Exporting Models from PyTorch — офіційна інструкція для експорту
в ONNX.
8. Transfer Learning vs. Feature Extraction — огляд різних стратегій.
9. What Is Transfer Learning? (IBM) — прикладне пояснення з точки зору
індустрії.
10.Best Practices for Transfer Learning (Papers with Code) — останні підходи та
дослідження.
