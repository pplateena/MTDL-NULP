Лабораторна робота №4
Розгортання моделі глибокого навчання як REST‑API з
використанням FastAPI, TensorRT та Docker
Мета роботи. Опанувати методи розгортання моделей глибокого навчання
за допомогою сучасних інструментів, реалізувати високопродуктивний
REST‑сервіс, оптимізувати інференс за допомогою ONNX Runtime/TensorRT,
забезпечити відтворюваність за допомогою контейнеризації.
Завдання
1. Oзнaйoмитися з oснoвними тeoрeтичними вiдoмoстями тa
рeкoмeндoвaнoю лiтeрaтурoю зa тeмoю рoбoти.
2. Взяти раніше натреновану модель з Лаб. 3, експортувати її у формат
ONNX. За потреби скористатись torch.onnx.export або еквівалентом у
TensorFlow.
3. Побудувати REST-API для інференсу моделі за допомогою FastAPI:
• реалізувати ендпойнт /predict (POST) для прийому зображення та
повернення результатів;
• реалізувати ендпойнт /health (GET) для перевірки статусу сервісу;
• використовувати Pydantic для валідації вхідних даних.
4. Забезпечити логіку інференсу через ONNX Runtime: використати CPU
або GPU-версію Runtime (за можливості); виконати оптимізацію графу
(graph_optimization_level); протестувати продуктивність (latency, throughput) на
серії запитів (50–100).
5. Створити Dockerfile для контейнеризації сервісу.
6. Побудувати образ та протестувати його запуск. Перевірити роботу API.
7. Оцінити продуктивність: порівняти latency для onnxruntime-cpu,
onnxruntime-gpu, TensorRT; побудувати таблицю результатів.
8. Оформити звіт. Структура звіту: титульний аркуш, мета роботи,
покроковий опис ходу роботи, приклад програмної реалізації, скріншоти з
демонстрацією результатів виконання програми, висновки.
Теоретичні ресурси
1. FastAPI Documentation (Official) — повна офіційна документація.

4. TensorRT Overview (NVIDIA) — концепти оптимізації інференсу,
підтримка FP16/INT8.
5. Docker for Beginners (Docker Docs) — офіційне введення в Docker.
6. Deploy ML Models with FastAPI + Docker (YouTube) — практичне відео з
поясненням крок за кроком.
7. ONNX Model Optimization (Medium) — гайд із оптимізації моделі.
8. Async and Await in Python (Real Python) — необхідно для розуміння
FastAPI.
9. Containerizing ML Models (Analytics Vidhya) — практичне пояснення.
10.CI/CD for ML with GitHub Actions — як автоматизувати деплой та тести.
